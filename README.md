# Imposter Party

A real-time multiplayer party game built with React, TypeScript, Express, and WebSockets.

## Run Locally

**Prerequisites:** Node.js (v18 or higher)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will start at `http://localhost:3000`

## How it works

- The game randomly selects a word from a pre-defined local list for each round.
- No external AI or API keys are required.

## Available Scripts

- `npm run dev` - Start the development server with hot reload
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Type check the TypeScript code
